mk.drawer demo

mk.drawer is a library agnostic UI wdget. You have to point to DOM element, that will be treated as a drawer. It will be wrapped by appropriate elements and remains in the same container

<div class='drawer-container-element'>
	some content can be placed here...

	<div id='drawerElement'>
		content of the drawer
	</div>
	<script>
		mk.drawer.create( document.getElementById('drawerElement') );
	</script>
</div>
